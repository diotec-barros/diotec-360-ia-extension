# Change Log

## [0.0.1] - Initial Release

### Features
- Generate code based on context
- Explain selected code
- Refactor code with AI suggestions
- Interactive chat with context awareness
- Support for Ollama and OpenAI providers
- Streaming responses
- MoE (Mixture of Experts) architecture with Writer + Critic
- SQLite-based memory system for interaction tracking
- Code actions for applying suggestions
- Configurable settings for all features

### Commands
- Generate
- Explain
- Refactor
- Chat
- Chat Clear History
- Apply Last Suggestion
- Configure OpenAI API Key
- Open Memory DB
