## Quick Start

### 1. Configure Your Identity
Before purchasing credits, you need to set up your Sovereign Identity:

1. Open VS Code Command Palette (`Ctrl+Shift+P` or `Cmd+Shift+P`)
2. Type: `DIOTEC 360: Configure Sovereign Identity`
3. Follow the prompts to generate your key pair
4. Your public key is your account identifier

### 2. Buy Credits
1. Open Command Palette
2. Type: `ANGO IA: Buy Credits`
3. Select a package:
   - **Starter**: 1,000 credits for $9.99
   - **Professional**: 6,000 credits for $49.99 (20% bonus)
   - **Enterprise**: 30,000 credits for $199.99 (50% bonus)
4. Confirm purchase
5. Complete payment in PayPal
6. Credits appear in your account within seconds

### 3. Check Balance
1. Open Command Palette
2. Type: `ANGO IA: View Balance`
3. See your current credit balance
4. Option to buy more credits

## Troubleshooting

### "Sovereign Identity not configured"
**Solution**: Run `DIOTEC 360: Configure Sovereign Identity` first

### "Failed to create PayPal order"
**Possible causes**:
- Server is down (check status)
- Invalid package selected
- Network connection issue

**Solution**: Try again in a few minutes

### "Credits not appearing"
**Possible causes**:
- Payment still processing (wait 1-2 minutes)
- Webhook delay
- Wrong public key

**Solution**: 
1. Check PayPal for payment confirmation
2. Wait 2 minutes
3. Refresh balance
4. Contact support if still missing

### "Balance shows 0 but I purchased"
**Solution**:
1. Verify payment in PayPal
2. Check you're using the same public key
3. Run `ANGO IA: View Balance` again
4. Contact support with transaction ID

## FAQ

### Q: Do credits expire?
**A**: No, credits never expire. Use them whenever you want.

### Q: Can I transfer credits to another account?
**A**: Not currently. Credits are tied to your public key.

### Q: What happens if I lose my private key?
**A**: Your credits are tied to your public key. If you lose your private key, you lose access to your credits. Keep it safe!

### Q: Can I get a receipt?
**A**: Yes, PayPal sends a receipt automatically. You can also download from PayPal dashboard.

### Q: Is there a free tier?
**A**: Yes! You can earn free credits by:
- Mining proofs (Logic Miner)
- Referrals (coming soon)
- Promotions and giveaways

### Q: Can I use credits on multiple devices?
**A**: Yes, as long as you use the same Sovereign Identity (public/private key pair).

### Q: What if the price changes?
**A**: Prices may change, but credits you already purchased remain valid at their original value.
