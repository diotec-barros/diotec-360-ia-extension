# 🚀 QUICK START - DIOTEC 360 IA v0.0.1

## ⚡ 5 Minutos para Começar

### 1. Reinicie o VS Code

```bash
# Feche o VS Code completamente
# Abra novamente ou pressione Ctrl+Shift+P > "Developer: Reload Window"
```

### 2. Configure sua API Key

#### OpenAI:

1. Pressione `Ctrl+Shift+P`
2. Digite: `DIOTEC 360 IA: Configure OpenAI API Key`
3. Cole sua chave da OpenAI
4. Pressione Enter

#### Anthropic (Claude):

1. Pressione `Ctrl+Shift+P`
2. Digite: `DIOTEC 360 IA: Configure Anthropic API Key`
3. Cole sua chave da Anthropic
4. Pressione Enter

### 3. Use o First Command

#### Generate Code:

1. Abra um arquivo TypeScript/Python/JavaScript
2. Selecione uma parte de código ou deixe vazio
3. Pressione `Ctrl+Shift+P` e digite: `DIOTEC 360 IA: Generate`
4. Veja a sugestão aparecer no painel!

#### Explain Code:

1. Selecione qualquer código
2. Pressione `Ctrl+Shift+P` e digite: `DIOTEC 360 IA: Explain`
3. Obtenha uma explicação detalhada

#### Chat:

1. Selecione código (opcional)
2. Pressione `Ctrl+Shift+P` e digite: `DIOTEC 360 IA: Chat`
3. Faça 🤖 perguntas sobre o código!

### 4. Todos os Comandos (Ctrl+Shift+P)

- **Generate** - Gera novo código
- **Explain** - Explica código selecionado
- **Refactor** - Refatora código selecionado
- **Chat** - Chat interativo com IA
- **Open Memory DB** - Ver banco de dados de memória
- **Configure OpenAI API Key** - Setup OpenAI
- **Configure Anthropic API Key** - Setup Claude
- **Configure Sovereign Identity** - Setup Z3 Judge
- **Buy Credits** - Comprar créditos
- **View Credit Balance** - Ver saldo

## ⚙️ Configurações Avançadas

### Mudar Provider (OpenAI → Claude → Ollama)

1. `File > Preferences > Settings`
2. Procure por: `angoIA.provider`
3. Escolha: `openai`, `anthropic`, ou `ollama`

### Ativar Modo Mixture of Experts

1. `File > Preferences > Settings`
2. Procure por: `angoIA.moe.enabled`
3. Ativar: `true`

### Desativar Mining

Se não quer minerar automático:

1. `File > Preferences > Settings`
2. Procure por: `angoIA.mining.enabled`
3. Desativar: `false`

## 🐛 Troubleshooting

### "Extension not loaded"

- Reinicie o VS Code
- Verifique se está em um workspace aberto

### "No API Key configured"

- Configure a chave em Settings → DIOTEC 360 IA
- Ou use `Ctrl+Shift+P` e procure por "Configure API Key"

### "Connection Error"

- Verifique internet
- Verifique se a API key é válida
- Se usar Ollama, verifique se está rodando em localhost:11434

### Commands não aparecem

- Recarregue: `Ctrl+Shift+P` > "Developer: Reload Window"
- Ou reinicie VS Code completamente
